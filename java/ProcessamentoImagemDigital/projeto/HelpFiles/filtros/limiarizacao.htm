Limiarização

Calculo de Limiarização é aquele onde após eleito/informado um valor como limiar,
percorre a imagem e verifica o valor do pixel onde caso o valor seja maior que o limiar,
seta o seu valor para 255 e caso seja menor, seta o seu valor para 0.
 Usando o programa:
Vá no menu Filtros e escolha a opção Limiarização

* Escolha o valor limiar

* Imagem Resultado
