Quantização

A quantização utiliza de um valor informado de quantizador, o que representa
a quantidade de cores que a imagem será representada, por exemplo:
quando informado 16, ele irá realizar a verificação do valor do pixel
e enquadrar em um dos 16 intervalor possiveis de cor entre 0 a 255,
com isso redefinindo a imagem para 16 tons de cor( quantizando a imagem ).

 Usando o programa:
Vá no menu Filtros e escolha a opção Quantização.

* Defina a quantidade de tons

* Imagem Resultado
